# Paola Alexandra Kabbalah — GitHub Pages

## Edita 1 solo archivo
`assets/js/config.js`
- `BASE_PATH` (debe ser `/paolaalexandrakabbalah`)
- `SUPPORT_EMAIL`
- `PRICES` (EUR/USD/MXN) — precios fijos por moneda
- `STRIPE_LINKS` — pega tus Payment Links por moneda (kit/basic/pro/premium)
- `GUIDE_LEAD_WEBHOOK_URL` (opcional: n8n)

## Foto (Sobre mí)
1) Sube tu foto como `assets/img/paola.jpg`
2) En `index.html`, busca `about-placeholder.svg` y cámbialo a `/assets/img/paola.jpg`

## Deploy (GitHub Pages)
Settings → Pages → Source: Deploy from a branch → Branch: `main` → Folder: `/ (root)`
